# 📚 EnglishCard

Приложение для изучения английского языка с использованием PostgreSQL и Streamlit.

## 📋 Описание

EnglishCard — это веб-приложение для изучения английских слов. Приложение позволяет:
- Тренировать перевод слов с русского на английский
- Добавлять собственные слова для изучения
- Отслеживать статистику обучения
- Персонализировать процесс обучения

## 🚀 Установка и запуск

### Требования
- Python 3.12+
- PostgreSQL 14+
- Git

### 1. Клонирование репозитория

```bash
git clone <https://github.com/KilmanR/english_card>
cd english_card
'''

###  2. Создание виртуального окружения

python3 -m venv .venv
source .venv/bin/activate  # Для Linux/Mac
# или
.venv\Scripts\activate  # Для Windows


### 3. Установка зависимостей

pip install -r requirements.txt


### 4. Настройка базы данных


# Создайте базу данных
sudo -u postgres psql

# В PostgreSQL выполните:
CREATE DATABASE english_card_db;
\q

# Инициализируйте таблицы (опционально, создадутся автоматически при запуске)
python3 -c "from models import init_db; init_db()"


###  5. Заполнение тестовыми данными (опционально)

sudo -u postgres psql -d english_card_db -f fill_data.sql

### 6. Запуск приложения

streamlit run main.py

Приложение откроется по адресу: http://localhost:8501

📖 Использование
Первый запуск

    При первом запуске введите ваше имя
    Нажмите "Начать изучение"

Тренировка слов

    Перейдите в раздел "🎯 Тренировка"
    Вам будет предложено перевести слово с русского на английский
    Выберите правильный вариант из 4 предложенных
    Нажмите "➡️ Следующее слово" для продолжения

Добавление слов

    Перейдите в раздел "➕ Добавить слово"
    Введите слово на английском и его перевод
    Нажмите "Добавить слово"

Удаление слов

    Перейдите в раздел "🗑️ Удалить слово"
    Выберите слово из списка
    Нажмите "Удалить"

Просмотр статистики

    Перейдите в раздел "📊 Статистика"
    Просмотрите:
        Общее количество слов
        Количество правильных ответов
        Процент точности
        Количество выученных слов

🗄️ Структура базы данных
Таблица users

    id — первичный ключ
    username — имя пользователя (уникальное)
    created_at — дата регистрации

Таблица words

    id — первичный ключ
    word_en — слово на английском
    word_ru — перевод на русский
    is_common — флаг общего слова (True) или личного (False)
    created_at — дата добавления

Таблица user_words

    id — первичный ключ
    user_id — внешний ключ на users
    word_id — внешний ключ на words
    is_correct — выучено ли слово
    attempts — количество попыток
    correct_answers — количество правильных ответов
    last_practiced — последняя практика

🛠️ Технологии

    Backend: Python 3.12+
    Web Framework: Streamlit
    Database: PostgreSQL
    ORM: SQLAlchemy
    Driver: psycopg2
